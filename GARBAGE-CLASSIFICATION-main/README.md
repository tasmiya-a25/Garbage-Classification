# 🗑️ Garbage Classification Using Deep Learning

## 📌 Overview

Garbage Classification is a deep learning project that automatically identifies the type of waste from an uploaded image using a Convolutional Neural Network (CNN). The system helps improve waste segregation by classifying garbage into predefined categories, making recycling and waste management more efficient.

The project includes a trained CNN model and a user-friendly web application where users can upload an image and instantly receive the predicted waste category.

---

## ✨ Features

* 📷 Upload garbage images for classification
* 🧠 CNN-based deep learning model
* ⚡ Real-time predictions
* 🌐 Clean and responsive web interface

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Garbage-Classification.git
```

### 2. Navigate to the project

```bash
cd Garbage-Classification
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

Open your browser and visit:

```text

```

## 🧠 Model

The garbage classification model is built using a **Convolutional Neural Network (CNN)**.

Typical workflow:

1. Load dataset
2. Preprocess images
3. Train CNN
4. Validate performance
5. Save trained model
6. Predict waste category from uploaded images
